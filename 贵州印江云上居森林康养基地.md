# 贵州印江云上居森林康养基地

## 一、基地基本情况概述

贵州印江云上居森林康养基地位于**铜仁市印江土家族苗族自治县峨岭街道大云村**，地处世界自然遗产地——**梵净山西麓**，2018 年被贵州省林业厅授予“省森林康养试点基地”。基地集**旅游度假、森林康养、休闲娱乐、茶文化体验、拓展训练、农耕体验、花木生产、状元特色餐、电子商务、研学教育**等功能于一体，是“书法之乡·养生印江”西线的重要节点。([携程][1])

2023 年起，基地又作为首批 **“省级森林康养基地”** 被贵州省林业产业联合会正式授牌，与石阡仙人街等基地一同构成贵州森林康养产业的重点样板。可见其在全省森林康养格局中的示范地位。参见贵州省林业局发布的《正式授牌｜贵州首批10家“省级森林康养基地”出炉》及相关报道。

## 指标 1：基地规模

* **评分：5 分**

* **事实依据：**

  * 你提供的汇总文档中，印江云上居森林康养基地核心区面积为 **670hm²**，属于 >400hm² 区间。
  * 该基地 2018 年被认定为省级森林康养试点基地，后又被授牌为首批省级森林康养基地，印证其用地规模与功能承载能力。参见贵州省林业局授牌公告：[正式授牌｜贵州首批10家“省级森林康养基地”出炉](https://lyj.guizhou.gov.cn/gzlq/slky/202301/t20230117_77953935.html)。 ([贵州省林业局][2])

* **评分依据：**

  * 指标 1 评分标准中，**基地面积 >400hm² 记 5 分**；670hm² 明显大于 400hm²，因此按 **5 分** 评价。

## 指标 2：景观生态资源独特性

* **评分：4 分**

* **事实依据：**

  * 基地位于**梵净山西麓**，依托世界自然遗产地梵净山的整体生态格局，区位于“原生态民俗森林康养区”与“林茶复合型”模式核心地带。([tongren.gov.cn][5])
  * 大云村林木苍翠，空气清新，是印江县首批新农村建设示范村，周边有高森林覆盖率茶园、梯田与传统村落，构成“茶园+森林+村落”的复合景观格局。参见携程游记与贵州政协报文章中对“云上居生态旅游观光示范地”的描述：

    * [乡游黔中·印江县 大云村 │ 走进梵净山深处，那座飘着茶香的国学村落](https://gs.ctrip.com/html5/you/travels/1446297/4039949.html) ([携程][1])
    * [千年紫薇王脚下的小镇风情](https://www.gzszx.gov.cn/gzzxb/web/doc/detail/d_1525674309320736) ([贵州省政协][3])
  * 梵净山整体是**国家 5A 景区 + 世界自然遗产 + 国家级自然保护区**，拥有黔金丝猴、珙桐等珍稀物种，“云上居”位于其西侧生态圈内部，虽不在保护区核心区，但共享其生态景观资源与品牌溢出效应。([8264.com][6])

* **评分依据：**

  * 基地本身主要以茶园景观、乡村景观、山地森林景观为主，缺少明确记录的**大量国家级珍稀物种**，但依托梵净山这一世界自然遗产与 5A 景区品牌，景观组合与生态环境独特性较突出。
  * 综合判断：符合“**有少量珍稀物种或景观特色较突出，与生态、旅游、康养相关资源获得国家级品牌或荣誉 1–2 项**”的情形，评 **4 分**。

## 指标 3：毗邻区域资源

* **评分：3 分**

* **事实依据：**

  * 大云村距印江县城约 **6.5km**，为梵净山西线游的节点乡村之一。([携程][1])
  * 梵净山景区西门为世界自然遗产梵净山的主要登山入口之一，是 5A 旅游景区核心资源。相关旅游线路与攻略均将“印江—梵净山西线”作为重要组合产品。([tongren.gov.cn][7])
  * 周边还有**团龙村（省级乡村旅游重点村）**、“大圣墩”等景区，构成“云上居 + 大云村 + 团龙村 + 梵净山西线”的文旅康养组团。([yp.gov.cn][8])

* **评分依据：**

  * 已与梵净山 5A 景区及多处乡村旅游点、康养资源形成线路化与品牌化的联动，但公开资料中能明确点名的 A 级景区与康养节点数量约为 **2–3 个**。
  * 更符合“**与当地文旅产品的融合较好，周边有 3A/4A 景区或其他旅游及康养资源 2–3 个**”的情形，因此暂按 **3 分** 保守评价（如后续官方补充更多周边资源清单，可上调）。

## 指标 4：康养规划

* **评分：4 分**

* **事实依据：**

  * 官方与媒体多次强调“云上居森林康养基地”集**旅游度假、森林康养、茶文化体验、拓展训练、农耕体验、花木生产、状元特色餐、研学教育**等于一体，体现了“农文旅康”复合功能布局。([携程][1])
  * 印江整体围绕“书法之乡·养生印江”“梵净西游记”品牌，对梵净山西线进行整体规划，云上居作为**西麓生态国际旅游度假区**链条中的重要节点，承担“茶旅康养+研学+乡村度假”的功能。([tongren.gov.cn][5])

* **评分依据：**

  * 基地规划能够体现明显的地域特色（书法文化、状元文化、茶文化、土家民族文化），森林康养区、游憩欣赏区、农文旅体验区与综合管理/居住服务区布局较为合理，康养产品具备较好的功能性与体验性。
  * 符合“**基地规划能体现地域特色，功能区布局较合理，康养产品功能与体验性较好**”的描述，评 **4 分**。

## 指标 5：酒店与公寓

* **评分：2 分**（偏保守，建议后续通过电话核实床位数）

* **事实依据：**

  * 公开资料显示，云上居属于**民宿+度假小屋集群**，以茶园树屋度假别墅、乡村民宿为主，强调“状元特色餐”“国学村落”“土家风情”等主题，环境与建筑风格与周边茶园、山林协调。([贵州省政协][3])
  * 未检索到官方公开的**总床位数**。从“民宿群”和“生态旅游观光示范地”体量推测，目前整体床位规模可能在 **100–300 张** 区间，尚不足 300–500 张的大型度假酒店集群规模。

* **评分依据：**

  * 室内外设计与环境协调性较好，有一定地域与主题特色，但在床位规模方面，按照**保守估计 100–300 张**处理，更接近“**床位 100–300 张，提供一定住宿能力**”的档位。
  * 因目前缺少官方精确统计数据，暂按 **2 分** 评定，并在备注中说明此项需通过与基地运营方电话核实后，可根据实际床位数进行修正。

## 指标 6：接待中心

* **评分：3–4 分（本次取 4 分）**

* **事实依据：**

  * 作为省级森林康养基地和热门乡村旅游目的地，“云上居”已形成较稳定的游客接待能力，媒体图片与文字多次出现**集中演出场地、活动广场、景观走廊、导览标识**等设施。([澎湃新闻][4])
  * 在“初夏康养之旅，学游共促发展”活动中，印江老年大学组织学员到“状元故里·云上居”开展康养学游体验，现场有**医生咨询健康问题、统一组织演出、集体活动区域、步道导览**等，说明基地具备一定的**综合接待与公共信息服务功能**。参见“[在‘游’中学 印江老年游学体验感满满~](https://www.tongren.gov.cn/2024/0521/318330.shtml)”。 ([tongren.gov.cn][9])

* **评分依据：**

  * 可确认至少有一处集中接待和活动组织区域，能够提供较好的公共信息与导览服务；是否存在 4–5 个分散服务点在公开资料中未被清晰描述。
  * 综合考虑，按“**设置至少一处接待中心，提供较好的公共信息与导旅讲解服务**”的标准，给予 **4 分**（如果后续确认仅为简易服务点，可调整为 3 分）。

## 指标 7：商务区

* **评分：3 分**

* **事实依据：**

  * 大云村与云上居周边发展了**茶叶、电商、农特产品销售**等业态，形成了以梵净山茶、长寿食品、土家特色农产品为主的乡村旅游购物场景。([携程][1])
  * 作为“生态旅游观光示范地”和“森林康养基地”，游客在此可以购买茶叶、农副产品、文创纪念品等，但尚未形成大规模、强主题的商业街区。

* **评分依据：**

  * 商业秩序与商品质量总体良好、价格合理，有一定地域特色（茶叶、土家文化、状元文化相关产品），但通过公开资料尚难认定“文化、地域与主题特色突出”的高等级标准。
  * 更接近“**商业秩序与商品质量良好、价格合理，特色一般**”的评价，故评分 **3 分**。

## 指标 8：健康服务中心

* **评分：3 分（信息不完全，偏保守）**

* **事实依据：**

  * 在老年游学活动报道中，明确写到学员在云上居**向梵净山医院医生咨询相关健康知识**，说明基地与专业医疗机构存在合作，能在基地内开展健康咨询服务。([tongren.gov.cn][9])
  * 省林业局及地方报道目前尚未公开云上居内是否建成面积 >2000m² 的独立“健康服务中心大楼”，更多是以**活动型、流动型健康服务（医生进基地）+ 森林康养环境**为主。

* **评分依据：**

  * 可以认为基地具有一定的健康咨询与康养服务功能，但从公开信息看，**专门的健康服务中心建筑面积及内部分区、设备情况未见详细披露**，保守判断总面积少于 2000m²。
  * 对照指标描述，暂按“**面积少于 2000m²，康养功能分区较明确、设施基本满足要求**”处理，给 **3 分**，并标注为“需与基地进一步核实建筑面积与功能分区”的指标项。

## 指标 9：康养步道 / 骑行道

* **评分：3–4 分（本次按 3 分保守计）**

* **事实依据：**

  * 云上居所在的大云村为“生态旅游观光示范地”，村内及周边建有**茶园观光步道、山地步道、景观走廊**等，适合森林漫步、观景、采茶体验。([携程][1])
  * 报道中多次出现“逶迤弯曲的景观走廊上散步观景、了解人文历史”，说明基地内部步行系统较完善，可将住宿区、活动区、茶园、农耕体验区等功能点串联。([tongren.gov.cn][9])
  * 目前公开资料未给出**步道精确里程**，但从“茶园+村落+森林”的空间尺度推测，主要康养步行线路长度大致在 3–5km 范围。

* **评分依据：**

  * 由于缺乏明确的公里数与康养体验点数量的官方说明，采用保守口径，按“**坡度较合适，长度 3–4km / 8–10km，有 4–5 个康养体验点**”的标准估计，初步评为 **3 分**。
  * 若后续基地提供精确数据（如康养步道总长 ≥5km 且体验点 ≥8 个），可调整为 4–5 分。

## 指标 10：外部交通环境

* **评分：2 分**

* **事实依据：**

  * 大云村距印江县城 **6.5km**，车程约 10–15 分钟，可通过县城道路快速抵达云上居。([携程][1])
  * 从杭瑞高速梵净山西收费站下高速，约 **30 分钟车程**可到达梵净山景区西门；云上居位于县城与西门之间，从高速出口到基地车程一般也在 **30–40 分钟以内**。([tongren.gov.cn][7])
  * 铜仁凤凰机场 / 铜仁高铁站到印江县城的综合接驳时间在 **1.5 小时左右**，再转乘旅游专线或车辆前往梵净山西线，整体时间明显超过 1 小时。([yp.gov.cn][8])

* **评分依据：**

  * 就三个“依托区”（高速出口、高铁站、机场）而言：

    * 高速出口 → 基地：约 0.5–0.7 小时 以内；
    * 高铁站/机场 → 基地：均 **>1 小时**。
  * 因此符合“**距离依托区（高速出口、高铁站、机场）有 1 处在 1 小时以内车程**”这一档，评分 **2 分**。

## 指标 11：内部交通服务设施

* **评分：3 分**

* **事实依据：**

  * 云上居以**步行体验**为主，已建设若干景观走廊、茶园步道和村内道路，方便游客在住宿区、活动区、观景点、茶园等之间步行往来。([携程][1])
  * 目前公开报道中未明确提到内部大规模环保电瓶车系统或系统性的无障碍设计，只能从部分照片及活动描述推测基础道路条件尚可。

* **评分依据：**

  * 可以认定基地内**车行道+人行道基本形成网络**，能支撑景区运营与游客步行；但由于缺少明确资料说明“全程无障碍设计”和“以公共型、共享型环保交通为主”等高标准内容，更接近：

    * “**内部交通网络车行道、人行道较为完善，但无环保交通**”
  * 故暂按 **3 分** 评价，后续如确认有系统性无障碍设施与环保交通工具，可提高至 4–5 分。

## 指标 12：中医康养特色项目供给

* **评分：2–3 分（本次取 2 分）**

* **事实依据：**

  * 老年游学活动中，**梵净山医院医生进驻云上居，为老年学员提供健康咨询**，说明基地与医疗机构合作开展一定的健康管理与康养服务。([tongren.gov.cn][9])
  * 公开资料中尚未看到“艾灸、推拿、拔罐、药浴”等系统化中医技术项目的详细清单，更多是“健康咨询+森林环境+太极等运动”的组合。

* **评分依据：**

  * 在缺乏明确文本支撑的情况下，保守认定基地目前已提供 **1–3 项与中医理念相关的康养服务（健康咨询、体质建议、饮食与作息指导等）**，尚达不到 4–5 种以上的中医特色项目规模。
  * 故暂按“**中医特色康养服务能力一般，提供 1–3 种中医康养项目**”的档位，给 **2 分**。如后续电话或官方材料能列出完整的中医康养项目清单，可重新评分。

## 指标 13：文化类项目供给

* **评分：4 分**

* **事实依据：**

  * 大云村兼具**书法文化**（严寅亮书法传统）与**“曹状元”状元文化**，云上居被定位为“飘着茶香的国学村落”，常态化开展国学、书法、乡村研学等活动。([携程][1])
  * 在“初夏康养之旅，学游共促发展”活动中，老年学员在云上居进行**传统乐器演奏、走秀表演、合唱等文艺节目**，并在景观走廊上**吟诗作赋**，体现较为丰富的文艺文化活动场景。参见：[在“游”中学 印江老年游学体验感满满~](https://www.tongren.gov.cn/2024/0521/318330.shtml)。 ([tongren.gov.cn][9])
  * 此外，大云村入选贵州省第二批乡村旅游重点村，常规的**乡村文化体验、非遗展示、农耕文化教育**也在此区域开展。([携程][1])

* **评分依据：**

  * 综合“书法+国学+状元文化+民俗展示+文艺演出+诗词创作+研学课程”等活动类型，文化类项目种类已明显超过 3–4 项，基本可达 **5–6 项以上**。
  * 因此更符合“**举办有科普、艺术创作展出活动、健康养生课堂等 5–6 项活动**”的标准，评 **4 分**。若后续有系统课程和长期活动清单，完全可以支持 5 分评价。

## 指标 14：体育类项目供给

* **评分：3–4 分（本次取 3 分）**

* **事实依据：**

  * 老年游学报道中明确提及：**健身操、太极、手杖操**等体育健身项目在云上居集中开展，参与人数多、组织形式成熟。([tongren.gov.cn][9])
  * 结合基地“森林康养+拓展训练+研学”的定位，可推断还会开展部分**拓展训练、徒步、简单山地运动**活动，只是公开报道未系统列举。([携程][1])

* **评分依据：**

  * 基于可确认的信息，稳定开展的体育类活动至少包括：**太极、健身操、手杖操 + 徒步散步**等，可保守视作 **3–4 项体育类项目**。
  * 因目前缺少骑行、专业拓展基地等详尽说明，先按“**体育类项目 3–4 项**”档给 **3 分**，后续如获得官方活动清单，可提高到 4 分甚至 5 分。

## 指标 15：药膳食疗类项目供给

* **评分：2–3 分（本次取 2 分）**

* **事实依据：**

  * 云上居周边强调“状元特色餐”“茶餐”等健康饮食体验，游客可在茶园民宿中品尝以梵净山茶、山间蔬菜为主的地方菜肴，兼具“清淡、少油、少盐”的养生特点。([携程][1])
  * 公开资料中并未看到针对**多类慢病人群定制药膳食疗方案**的正式描述，更多是从食材健康、营养搭配层面体现“养生饮食”理念。

* **评分依据：**

  * 可认定基地在饮食上具有一定的“**养生膳食**”特点，适合一般人群与中老年、亚健康人群，但针对多种慢病人群的专业化药膳服务尚无明确信息支撑。
  * 因此暂按“**能为一般人群、亚健康人群、重点人群提供一定药膳/养生餐饮**”的水平处理，评 **2 分**；如后续能从基地获取具体药膳菜谱与针对不同病种人群的配方清单，可上调到 3–4 分。

## 指标 16：户外康养体验项目供给

* **评分：4 分**

* **事实依据：**

  * 结合多篇报道及旅游宣传，云上居及其所在区域已形成多样化户外康养体验项目，主要包括：

    1. **森林散步与茶园漫步**（景观走廊、林中小径）；([携程][1])
    2. **茶文化体验**：采茶、制茶、品茶；([携程][1])
    3. **农耕体验**：参与农事、体验乡村劳作；
    4. **拓展训练与研学活动**：面向青少年与团队的户外拓展；([澎湃新闻][4])
    5. **太极、健身操、手杖操等户外健康运动**；([tongren.gov.cn][9])
    6. **摄影、观景、静心休憩**等轻量户外活动。
  * 若将不同体验类型细分，其项目数基本可达到 **7–9 个** 左右。

* **评分依据：**

  * 对照户外康养体验项目供给指标，云上居已经具备**较丰富且特色鲜明的户外康养场所与项目组合**，项目类型多于 6 项，接近“7–9 个”的档位。
  * 因此评定为“**有特色鲜明的户外康养体验场所 7–9 个**”，给 **4 分**。
